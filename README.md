    * **petal:**  بتلة
    * **sepal:**  سبال

* **علم الحيوان:**

    * **recessive:**  متنحي

    * **heterozygous:**  متغاير الجينات
    * **mutation:**  طفرة

    * **immunis:**  المناعي

* **علم البيئة:**
    * **biosphere:**  المحيط الحيوي
    * **ecosystem:**  النظام البيئي
    * **community:**  المجتمع
    * **population:**  الجماعة
    * **habitat:**  الموئل
